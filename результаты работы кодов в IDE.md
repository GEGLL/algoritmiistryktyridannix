1. Линейный поиск (Python)

Перебирает массив поэлементно

Сравнивает каждый элемент с искомым значением

Возвращает индекс при нахождении или -1 при отсутствии

2. Пирамидальная сортировка (Java)

Преобразует массив в бинарную кучу

Последовательно извлекает максимальные элементы

Размещает их в конце массива в отсортированном порядке

3. Сортировка вставками (C++)

Постепенно строит отсортированную часть массива

Для каждого нового элемента находит правильную позицию

Сдвигает элементы для освобождения места под вставку

4. Интерполяционный поиск (Java)

Использует формулу интерполяции для предсказания позиции

Работает только с отсортированными массивами

Эффективен при равномерном распределении данных

5. Быстрая сортировка (C++)

Выбирает опорный элемент (pivot)

Разделяет массив на элементы меньше и больше опоры

Рекурсивно сортирует полученные части

6. Сортировка выбором (C++)

Находит минимальный элемент в неотсортированной части

Обменивает его с первым элементом неотсортированной части

Повторяет для оставшейся части массива

7. Бинарный поиск (C++)

Работает с отсортированными массивами

На каждом шаге делит область поиска пополам

Эффективно находит элемент за логарифмическое время

8. Сортировка пузырьком (Python)

Многократно проходит по массиву

Сравнивает и обменивает соседние элементы

"Всплывает" наибольшие элементы к концу массива

9. Поиск Фибоначчи (Python)

Использует числа Фибоначчи для определения точек сравнения

Работает с отсортированными массивами

Обеспечивает стабильную логарифмическую сложность

10. Сортировка Шелла (Python)

Выполняет сортировку вставками с убывающими шагами

Начинает с больших расстояний между сравниваемыми элементами

Постепенно уменьшает шаг до 1 для финальной сортировки

11. Сортировка слиянием (Java)

Рекурсивно делит массив на подмассивы

Сортирует каждый подмассив

Объединяет отсортированные подмассивы в один
